Case Studies

FreeTherapy cpt is a cross-platform tool (fictional) to help communities ensure all residents have mental health resources in Cape Town. Both the app and website allow users to learn about mental health and book free counseling sessions. The main goal of this case study was to design for mobile first with an application and then build a website based off of the design to be used seamlessly with the app. This project was made using Figma.

The Goal of this case study was to create a website counterpart for the LocalCafe app that is on par with competitors in order to bring in more customers and increase customer retention.
